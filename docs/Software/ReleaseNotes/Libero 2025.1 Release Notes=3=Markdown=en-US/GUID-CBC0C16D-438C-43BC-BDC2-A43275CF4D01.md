# Identify ME Tool Upgrade

Libero SoC v2025.1 supports Identify ME version V-2023.09M-5, which introduces additional<br /> support for RTG4 Pre-arm trigger. This version provides debug support for the following<br /> devices as well:

-   RT PolarFire
-   RT PolarFire SoC
