# Using the PolarFire SoC MSS Configurator GUI

The PolarFire SoC MSS Configurator GUI has the following tabs.

-   Peripherals
-   DDR Memory
-   L2 Cache
-   Crypto
-   MSS to/from Fabric Interface Controllers
-   Clocks
-   MSS I/O Attributes
-   Memory Partition and Protection
-   Misc
