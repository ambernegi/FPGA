# smartpower\_compute\_vectorless

## Description

This Tcl command executes a vectorless analysis of the current operating mode.

```
smartpower_compute_vectorless
```

## Arguments

|Parameter|Type|Description|
|---------|----|-----------|
|None|None|None|

|Return Type|Description|
|-----------|-----------|
|None|None|
