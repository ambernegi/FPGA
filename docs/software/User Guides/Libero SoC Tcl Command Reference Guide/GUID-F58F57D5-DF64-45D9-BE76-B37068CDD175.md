# get\_current\_scenario

## Description

Returns the name of the current timing scenario.

```
get_current_scenario
```

## Arguments

|Parameter|Type|Description|
|---------|----|-----------|
|`None`|None|None|

|Return Type|Description|
|-----------|-----------|
|`string`|Name of the current timing scenario.|
