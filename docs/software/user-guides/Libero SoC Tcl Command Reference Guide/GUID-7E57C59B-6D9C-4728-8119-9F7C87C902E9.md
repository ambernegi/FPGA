# ungroup

## Description

This Tcl command disassociates the probes as a group.

```
ungroup -name {group name}
```

## Arguments

|Parameter|Type|Description|
|---------|----|-----------|
|name|string|Specifies name of the group.|

|Return Type|Description|
|-----------|-----------|
|None|None|

## Error Codes

|Error Code|Description|
|----------|-----------|
|None|None|
