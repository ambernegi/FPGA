# list\_false\_paths

## Description

Returns<br /> details about all of the false paths in the current timing constraint scenario.

```
list_false_paths
```

## Arguments

|Return Type|Description|
|-----------|-----------|
|`string`|Returns details about all of the false paths in the current timing constraint scenario.|
