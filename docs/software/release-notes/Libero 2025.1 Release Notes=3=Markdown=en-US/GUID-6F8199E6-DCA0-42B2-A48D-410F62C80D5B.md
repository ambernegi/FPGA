# Octal DDR PHY DRC Limit Update

Starting with the Libero v2025.1 release, the FMAX for OCTAL\_SPI IOD interfaces is increased from<br /> 400 Mbps to 500 Mbps for STD and -1 for all the GPIO and HSIO banks.
