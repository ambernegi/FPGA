# Low-Voltage TTL \(LVTTL\)

LVTTL is a general-purpose standard \(EIA/JESD8-B\) for 3.3V applications.<br /> It uses an LVTTL input buffer and a push-pull output buffer. GPIO supports the LVTTL I/O<br /> standards, and the LVTTL output buffer can have up to six different programmable drive<br /> strengths.
